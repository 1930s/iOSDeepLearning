## 建这个仓库的目的

世人皆说阅读开源框架的源代码对于功力有显著的提升，然则笔者阅读开源框架之时，每每遇到一些底层知识的时候便一头雾水，无法再阅读下去，故打算先全面复习一下iOS的底层原理知识以做储备，在这里将自己学习iOS底层知识的心得记录下来，希望能对后续的阅读开源框架有所帮助。我会不断更新这个仓库中的文章，如果想要关注可以点 star。
